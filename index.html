Tokyo Trains

I created a dataset of every train station in Tokyo, Japan. Using R, I created an interactive map visualizing these 600+ stations. I used this visualization to tell a story about the privatization of transportation in Japan.

Data Curation

Using open access data from PsychArchives, I created a repository on GitHub with curated data of a dataset. This includes dataset files, the dataset metadata in XML according to DataCite standards, a readme, and a final report.
